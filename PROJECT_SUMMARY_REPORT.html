1. Project Overview

The Juan365 Creative Ticketing System is a comprehensive task management platform designed specifically for the Creative team. It streamlines the workflow from ticket creation to completion with a two-step approval process, real-time notifications, and detailed analytics.

Visual overview of all major pages and features in the ticketing system.

User Authentication

Secure login with username/password. Account lockout after failed attempts. New users can register and await admin approval.

2.2 Dashboard

Main Dashboard

Displays ticket statistics by status (Total, Pending Approval, In Progress, Completed), priority breakdown, weekly trends, and user's task list with quick actions.

juan365-ticketing-frontend.vercel.app/tickets

All Tickets View

Paginated table with search, filters (status, priority, department), and inline action buttons. Supports table/card view toggle. Shows 227+ tickets with real-time status updates.

2.4 Create Ticket

juan365-ticketing-frontend.vercel.app/tickets/new

New Ticket Form

Create tickets with title, description, department, product, request type (Socmed Posting, Ads, Telegram, Videoshoot, etc.), priority with auto-calculated deadline, file attachments, and optional pre-assignment to Creative team members.

2.5 Ticket Preview Modal

Quick Preview Modal

Quick Ticket Preview

Click any ticket row to see a quick preview with key details (status, priority, requester, assignee, deadline, products) and action buttons without leaving the list view.

2.6 Ticket Detail

juan365-ticketing-frontend.vercel.app/tickets/337

Full Ticket Details

Complete ticket information including description, comments thread, file attachments, assigned team members, ticket history with rollback capability, and all available actions based on status and user role.

2.7 Analytics Dashboard

juan365-ticketing-frontend.vercel.app/analytics

Performance Analytics

Comprehensive metrics including team performance, processing times, output quantities, criteria breakdown (Image/Video), Ads and Telegram product output, request type distribution, and date-filtered reporting.

2.8 User Management

juan365-ticketing-frontend.vercel.app/users

Admin User Management

Manage all users with role assignment (Member, Approver, Admin), department assignment, account status (Active/Locked/Inactive), password reset, and user approval workflow for new registrations.
